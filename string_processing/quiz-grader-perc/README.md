<h1>Basic Quiz Grader Program: Percentage System</h1>
<p>This program gives out a grade for a quiz. The grading system is based off of a percentage system:</p>
<ul>
    <li>A: 90% - 100%</li>
    <li>B: 80% - 89%</li>
    <li>C: 70% - 79%</li>
    <li>D: 60% - 69%</li>
    <li>F: 0% - 59%</li>
</ul>
